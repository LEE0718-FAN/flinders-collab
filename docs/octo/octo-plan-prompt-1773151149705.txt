## Project Overview

This project is a **team project management app for university students**.

The app allows students to:

* log in using their university ID
* create or join a team room
* manage schedules and deadlines
* upload and share files
* chat in real time
* share live location when team meetings are scheduled
* reduce lateness and improve team coordination

The product should feel modern, simple, fast, and reliable for student use.

---

## Core Product Goal

Build a collaborative university team app where students can manage group projects in one place instead of switching between multiple tools such as messaging apps, cloud drives, calendars, and map apps.

The app should focus on:

* clear team communication
* schedule visibility
* accountability
* easy file sharing
* practical location sharing for offline meetings

- Therefore: each agent must own exclusive files. NO overlap allowed.
